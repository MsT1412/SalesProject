# Retail Sales Analytics & Business Intelligence Pipeline

## Project Overview
This project presents an end-to-end data analytics pipeline built using a retail sales dataset. It demonstrates the full workflow from raw data ingestion and SQL-based data cleaning to exploratory data analysis and executive-level visualisations in Python.

The goal of this project is to simulate a real-world data analyst workflow and extract meaningful business insights that support decision-making.
_________________________________________________________________________________________
## Objectives
This project answers key business questions:

- How is the business performing overall?
- Which regions generate the most sales and profit?
- Which product categories are most profitable?
- Which products drive revenue and which reduce profitability?
- How do discounts impact profit?
- Which customer segments contribute the most value?
- Are there trends or patterns in sales over time?

## Project Workflow

### 1. Data Ingestion
- Imported raw dataset into MySQL
- Structured the database for analysis

### 2. Data Cleaning (SQL)
- Created a cleaned dataset from raw data
- Converted date formats
- Standardised numeric fields
- Removed inconsistencies and validated data

### 3. Data Validation
- Verified row counts
- Checked minimum and maximum values
- Assessed profit distribution
- Ensured data reliability

### 4. Exploratory Data Analysis
Performed analysis using SQL and Python:

- KPI summary (sales, profit, orders)
- Regional performance analysis
- Category and sub-category profitability
- Product-level performance
- Customer segment analysis
- Discount vs profit relationship
- Monthly sales trends
___________________________________________________________________________________________
## Key Insights

- The **West region** generates the highest total sales, making it the strongest revenue contributor.
- The **Technology category** delivers the highest profit, while **Furniture** shows lower profitability.
- A small number of products drive a large portion of total revenue.
- Several products generate negative profit, indicating pricing or discount inefficiencies.
- Higher discounts are associated with lower profitability.
- The **Consumer segment** contributes the highest sales and profit.
- Sales trends indicate potential seasonality in business performance.
